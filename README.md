# Push Swap Input Generator
This is an input generator for 42`s push_swap :)

## Usage
Run the script
```bash
$ make
```
If you want to change the amount of total numbers or the max number
```bash
$ make TOTAL_NUMBERS=<number> MAX_RANDOM=<number>
```

## Defaults
**TOTAL_NUMBERS**
500

**MAX_RANDOM**
500
